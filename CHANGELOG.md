# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [Unreleased]

### Added
- Reorganized project structure with `src/`, `include/`, `scripts/` directories
- Added comprehensive documentation for open source contributions
- Added GitHub Actions workflows for CI/CD
- Added issue and pull request templates

## [1.0.1] - 2024-12-04

### Added
- Initial stable release
- Real-time CPU usage and temperature monitoring
- Memory and swap usage tracking
- System uptime and load average display
- CPU clock speed and voltage monitoring
- CSV logging capabilities
- Native GTK3 interface
- Debian package support

### Fixed
- Performance improvements
- UI refinements

## [1.0.0] - 2024-12-03

### Added
- Initial release
- Basic system monitoring functionality
- GTK3 GUI implementation
- Raspberry Pi support

- **Added** for new features
- **Changed** for changes in existing functionality
- **Deprecated** for soon-to-be removed features
- **Removed** for now removed features
- **Fixed** for any bug fixes
- **Security** in case of vulnerabilities
